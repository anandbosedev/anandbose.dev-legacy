BetterJson

class BetterJson(tag: String, root: JSONObject, logCallback: (tag: String, msg: String) -> Unit?)
Content copied to clipboard

BetterJson class wraps a JSONObject allows to access a target object or value with path. BetterJson caches the nested objects for performance.

Parameters

tag

A tag for generating logs

root

The root JSONObject. BetterJson assumes the JSONObject is not altered after wrapping with BetterJson class. If the JSONObject is mutated, BetterJson cannot track the changes, may leads to erroneous cached values.

logCallback

Callback for method to handle logs
